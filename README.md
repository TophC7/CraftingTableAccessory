# Crafting Table Accessory

Equip a crafting table as an accessory to craft anywhere.

## How it works

- Equip a crafting table in the dedicated accessory slot
- A button appears in your inventory screen — click it to open a 3x3 crafting grid
- Unequipping the table closes the crafting grid immediately
- Optional keybind to open crafting from anywhere (unbound by default)

## Why?

Some mods use complicated backpacks or wands for portable crafting. My favorite solution was always the original [Inventory Crafter](https://www.curseforge.com/minecraft/mc-mods/inventory-crafter), but keeping a crafting block in your inventory takes up space. 

This mod uses the [Accessories](https://modrinth.com/mod/accessories) API to give the crafting table its own special equipment slot, letting you craft on the go without sacrificing an inventory slot.

## Compatibility

- Requires [Accessories](https://modrinth.com/mod/accessories)
- Requires NeoForge  1.21.1 (21.1.219+)
- Don't use this and [Inventory Crafter Returns](https://www.curseforge.com/minecraft/mc-mods/inventory-crafter-returns) together something will probably break; plus they do the same thing.
